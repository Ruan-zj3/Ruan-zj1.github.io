Reflection on the Use of GenAI

1. GenAI Tool I Used

For this assessment, I used ChatGPT (GPT-5 Mini). I chose it because it provides clear explanations, can generate code snippets, and helps organize ideas efficiently.

2. How GenAI Helped Me

GenAI assisted me in several ways during this assessment:

  • It helped generate the **full website structure** including HTML, CSS, and JavaScript code.
  • Provided **sample visualizations** using Chart.js for my Data Visualization page.
  • Suggested **content and writing** for reflection and about pages, making it easier to express my ideas clearly.

Example: Instead of struggling with JavaScript syntax for a scatter plot, GenAI provided a ready-to-use script with numerical data points and labels.

3. Benefits of Using GenAI

The tool is beneficial because:

  • It saves **time and effort** in coding, formatting, and generating content.
  • It offers **examples and explanations**, which help me understand concepts faster.
  • It supports **creative ideas** and provides suggestions I might not have thought of.

4. Challenges of Using GenAI

Despite the benefits, there are some challenges:

  • Sometimes the output may need **manual editing** to fit my exact requirements.
  • It may produce **generic content** that requires personalization to reflect my own voice.
  • It cannot always check for **local files or personal data**, so I have to manually link images or videos.

5. Future Use of GenAI

I believe GenAI tools will assist me in my future studies and work by:

  • Generating **prototype code** and examples for projects.
  • Helping with **research summaries and idea organization**.
  • Supporting **creative problem-solving**, especially for complex tasks where brainstorming is needed.

6. Other Thoughts

Using GenAI has shown me the power of AI as a collaborative assistant. While it cannot replace human reasoning and creativity, it is an excellent tool to enhance productivity, learning, and clarity in projects.
